## Dictionary

Props - data passed through components

Componsition - organizing react components in the most optimal way 

Hooks - Functions starting with use are called Hooks. They are functions that return js variants. Handle state 

Utils - Helper functions that are used across multiple components or modules within a React application

SPA - single page application. There's only one page and the javascript changes the content

SSR - Server side rendering

JSX - mix of html and typescript
A JSX element is a combination of JavaScript code and HTML

Fragments - <> and </>

Redux - An state manager

TypeScript - TypeScript is a way to add type definitions to JavaScript codebases

Virtual DOM = VDOM - React search for differences when state changes and do the "reconciliation" that is update the DOM with the changes

Is a Javascript object

Only updates the section of the page where the change was made. Not the entire page

Promise - Object which represents the completion of an asynchronous operation and its return value

A reducer in Redux is a pure function that describes how the application state should be updated in response to an action.
